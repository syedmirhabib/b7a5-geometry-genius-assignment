1. What is DOM and what is the purpose of DOM?

The Document Object Model is called DOM. it's a programming interface for web documents. it's generate an objects of Nodes (like a tree) and represents to a web page. The main purpose of using DOM is that - to make a website in dynamic and interactive.

2. How will you select HTML elements using DOM? Name the DOM methods.

We can select HTML elements with dom in many available methods. here are the some methods:

  • getElementById():

    This method selects an elements from its 'id' attribute.

  • getElementsByClassName():

    This method selects a collection of HTML elements from their 'class' attribute.

  • getElementsByTagName():

    This method selects a collection of HTML elements from their 'tag name'.

  • querySelector():

    This method selects an elements from its 'CSS selector'.

  • querySelectorAll():

    This method selects a collection of NodeList from their 'CSS selector'.

3. What is event bubble?

Event bubble is a functionality or mechanism in DOM. its worked when an event that is triggered on an element is also propagated to its parent elements in the DOM tree.

4. What is a callback function and why will you use it?

A callback function is a function which is an argument of another function. when a specific event occurs callback function is invoked automatically. generally callback function are used for asynchronous behavior. we can also used for reuseable code, easier to maintain, handling error and much more.
